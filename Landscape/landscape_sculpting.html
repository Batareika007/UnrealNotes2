Landscape Sculpt — редактирование рельефа

Режим Sculpt используется для создания рельефа: гор, холмов, долин, дорог, рек, ям и любых других форм поверхности. В UE5 Sculpt работает вместе с Edit Layers, что позволяет редактировать рельеф безопасно и гибко.

Основные элементы интерфейса

  • Tool — выбор инструмента редактирования.
  • Brush — выбор типа кисти.
  • Falloff — форма и мягкость краёв кисти.
  • Strength — сила воздействия инструмента.

Инструменты Sculpt

Sculpt

Основной инструмент для поднятия и опускания рельефа.

  • ЛКМ — поднимает поверхность.
  • Shift + ЛКМ — опускает поверхность.
  • Tool Strength — скорость изменения рельефа.
  • Use Clay Brush — добавляет материал только там, где поверхность ниже текущей высоты (удобно для гор).

Smooth

Сглаживает острые углы и неровности. Используется для финальной обработки рельефа.

Flatten

Делает поверхность ровной, выравнивая её под одну высоту. Первая точка, по которой вы кликнули — задаёт уровень выравнивания.

Ramp

Создаёт наклонную поверхность между двумя точками. Используется для дорог, подъёмов, спусков.

Noise

Добавляет случайный рельеф. Можно настроить:

  • Add — добавляет шум.
  • Sub — вырезает шум.
  • Scale — размер шума.

Visibility

Делает отверстия в ландшафте (например, для входа в пещеру). Требует материала с поддержкой Masked и Landscape Visibility Mask.

Select

Позволяет выделять части ландшафта и копировать их в виде заготовки (Stamp). Полезно для повторяющихся элементов рельефа.

Mirror

Создаёт зеркальную копию выбранной части ландшафта. Используется для симметричных уровней.

Устаревшие инструменты (не рекомендуется использовать в UE5)

Эти инструменты остались с времён UE4 и работают плохо или нестабильно:

  • Erosion — старый алгоритм эрозии.
  • Hydro Erosion — имитация водной эрозии, даёт артефакты.
  • Retopologize — почти не используется, не даёт ощутимой пользы.

В UE5 вместо них рекомендуется использовать Landmass Brushes (см. раздел 5).

Типы кистей (Brush Types)

Circle

Стандартная круглая кисть.

Alpha Brush

Позволяет использовать собственные текстуры как форму кисти. Отлично подходит для создания тропинок, скал, отпечатков рельефа.

Pattern

Рисование по шаблону (трафарету). Используется для повторяющихся форм.

Component

Воздействует сразу на целый компонент ландшафта. Полезно для крупных изменений.

Falloff — мягкость краёв кисти

Определяет, насколько плавно инструмент воздействует на края кисти.

  • Linear — резкий край.
  • Smooth — мягкий переход.
  • Spherical — естественная форма для холмов.

Blueprint Brushes (Landmass)

В UE5 можно использовать процедурные кисти через Landmass Plugin:

  • Mountain
  • Valley
  • Road
  • River
  • Noise

Эти кисти автоматически создают плавные формы и работают через Edit Layers, что делает их безопасными и удобными.

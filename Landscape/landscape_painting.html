Landscape Paint — раскраска ландшафта

Режим Paint используется для нанесения материалов на поверхность ландшафта: трава, земля, камни, песок, снег, грязь и любые другие типы покрытия. В UE5 система раскраски работает через слои материала и поддерживает автоматическую генерацию травы, смешивание по высоте и Runtime Virtual Textures.

Как работает раскраска ландшафта

Ландшафт раскрашивается с помощью Landscape Layers, которые задаются в материале. Каждый слой — это отдельная текстура или набор текстур, которые можно наносить кистью.

Создание материала для ландшафта

Чтобы начать раскрашивать ландшафт, нужно создать материал:

  1. Создайте новый Material.
  2. Добавьте узел LandscapeLayerBlend.
  3. Через «+» добавьте нужные слои (Grass, Dirt, Rock и т.д.).
  4. Для каждого слоя задайте:
    • Base Color (Diffuse)
    • Normal
    • Roughness
  5. Примените материал к Landscape.

Типы смешивания слоёв (Blend Types)

Weight Blend

Стандартный режим. Новый слой заменяет предыдущий. Подходит для большинства материалов.

Alpha Blend

Слой накладывается поверх других. Если удалить слой — можно увидеть нижний.

Height Blend

Смешивание по высоте. Позволяет делать:

  • камни поверх земли,
  • траву у основания скал,
  • плавные переходы между материалами.

Это самый реалистичный способ смешивания материалов.

Layer Info Objects

Чтобы слой можно было рисовать, нужно создать Layer Info:

  • Weight-Blended Layer — создаёт текстуру веса.
  • Non-Weight-Blended Layer — используется для масок (например, камни).

Без Layer Info слой не появится в режиме Paint.

Инструменты Paint

Инструменты такие же, как в Sculpt:

  • Paint — наносит материал.
  • Erase — удаляет материал.
  • Smooth — сглаживает переходы между слоями.
  • Brush Size — размер кисти.
  • Brush Falloff — мягкость краёв.

Auto-Materials (автоматические материалы)

Современные материалы для ландшафта могут автоматически менять покрытие в зависимости от:

  • угла поверхности (Slope)
  • высоты (Height)
  • маски (RVT)

Например:

  • трава на ровных участках,
  • камни на крутых склонах,
  • грязь в низинах.

Runtime Virtual Texture (RVT)

RVT — это виртуальная текстура, которая позволяет:

  • делать плавные переходы между ландшафтом и объектами,
  • создавать следы на земле,
  • передавать цвет и нормали ландшафта в траву,
  • делать реалистичное смешивание материалов.

RVT — современный стандарт для UE5 и рекомендуется для всех проектов.

Landscape Grass Output

Позволяет автоматически генерировать траву, кусты и мелкие объекты на основе слоёв материала.

Например:

  • на слое Grass появляется трава,
  • на слое Dirt — ничего,
  • на слое Rock — мох или кусты.

Это намного эффективнее, чем вручную размещать траву через Foliage Tool.

Советы по оптимизации

  • Используйте Height Blend для реалистичных переходов.
  • Не создавайте слишком много слоёв (8–12 — оптимум).
  • Используйте RVT для травы и смешивания материалов.
  • Храните текстуры в 2K или 4K, но не выше.

Landscape — основы работы с ландшафтом

Landscape — это система создания больших игровых территорий в Unreal Engine. Она состоит из компонентов, секций и слоёв, которые позволяют формировать рельеф, раскрашивать поверхность и размещать растительность.

Три основных режима Landscape

  • Manage — создание и управление ландшафтом.
  • Sculpt — редактирование рельефа: горы, холмы, ямы, дороги.
  • Paint — раскрашивание поверхности: трава, камни, земля, снег.

Manage — создание и управление ландшафтом

Основные параметры при создании ландшафта:

  • Section Size — количество полигонов в одной секции (от 7×7 до 255×255).
  • Sections per Component — количество секций внутри компонента (1×1, 2×2).
  • Number of Components — сколько компонентов будет в ландшафте.
  • Overall Resolution — итоговое разрешение ландшафта в полигонах.
  • Total Components — общее количество компонентов.
  • Fill World — автоматически заполняет весь мир ландшафтом.
  • Scale — масштаб ландшафта. По умолчанию 100 = 1 метр на полигон.
  • Create — создание ландшафта.

После создания ландшафта рекомендуется сразу перейти в режим Sculpt.

Управление существующим ландшафтом

  • Selection — выбор отдельных компонентов.
  • Add/Delete — добавление или удаление частей ландшафта.
  • Move to Level — перенос компонентов в другие уровни (для World Partition или больших карт).
  • Change Component Size — изменение размера секций (не рекомендуется в UE5, лучше пересоздать).

Edit Layers (важно!)

В UE5 рекомендуется включать Edit Layers. Это позволяет редактировать рельеф слоями, как в Photoshop:

  • отдельный слой для гор,
  • отдельный слой для дорог,
  • отдельный слой для рек,
  • отдельный слой для ручного скульптинга.

Это делает работу безопасной: можно отключать, менять порядок и удалять слои без разрушения рельефа.

Landmass Plugin (современный способ создавать дороги и реки)

Вместо старого инструмента Edit Splines рекомендуется использовать Landmass:

  • Mountain Brush
  • Valley Brush
  • Road Brush
  • River Brush
  • Noise Brush

Landmass автоматически деформирует ландшафт, создаёт плавные переходы и работает через Edit Layers.

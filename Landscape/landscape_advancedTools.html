Advanced Landscape Tools — продвинутые инструменты ландшафта

Unreal Engine 5 предоставляет набор современных инструментов для создания больших, реалистичных и оптимизированных миров. Эти инструменты работают поверх стандартного Landscape и позволяют создавать реки, дороги, горы, океаны, процедурную растительность и огромные территории с высокой производительностью.

Landmass Plugin — процедурные кисти для рельефа

Landmass — это современная система процедурного редактирования ландшафта. Она работает через Edit Layers и позволяет создавать рельеф автоматически.

Основные Landmass Brushes

  • Mountain — создаёт горы и возвышенности.
  • Valley — формирует долины и углубления.
  • Road — создаёт дороги, тропы, трассы.
  • River — формирует русла рек.
  • Noise — процедурный шум для естественного рельефа.

Преимущество Landmass — рельеф всегда получается плавным, корректным и легко редактируется.

Water System — реки, озёра, океаны

Water System — это мощный инструмент UE5 для создания водоёмов. Он автоматически взаимодействует с ландшафтом и создаёт реалистичную воду.

Типы воды

  • River — реки с течением.
  • Lake — озёра и пруды.
  • Ocean — бесконечный океан.
  • Custom Water Body — пользовательские формы.

Вода автоматически вырезает русло в ландшафте, если включён Landmass.

Virtual Heightfield Mesh (VHM)

VHM позволяет отображать огромные ландшафты с высокой детализацией, используя виртуальные текстуры вместо геометрии.

Преимущества:

  • идеально для больших миров;
  • высокая производительность;
  • поддержка Nanite;
  • поддержка RVT;
  • можно использовать для скал, утёсов, больших форм.

VHM часто используется вместе с World Partition.

Nanite Landscape (UE5.3+)

В UE5.3 появилась экспериментальная поддержка Nanite для ландшафта. Это позволяет использовать миллионы полигонов без потери производительности.

Преимущества Nanite Landscape:

  • не нужны LOD;
  • идеально для скал, каньонов, гор;
  • высокая детализация на больших расстояниях;
  • отлично работает с VHM.

Пока что Nanite Landscape не поддерживает все функции Paint, поэтому используется в гибридных проектах.

Runtime Virtual Texture (RVT)

RVT — ключевой инструмент UE5 для смешивания материалов и передачи данных между ландшафтом и объектами.

Что можно делать с RVT

  • плавно смешивать скалы и землю;
  • делать следы на поверхности;
  • передавать цвет и нормали в траву;
  • делать мягкие переходы между объектами и ландшафтом;
  • создавать маски для PCG и Foliage.

RVT — стандарт для всех современных проектов.

PCG (Procedural Content Generation)

PCG — новая система процедурной генерации контента в UE5. Она заменяет старые инструменты Foliage и позволяет создавать сложные экосистемы.

Что можно генерировать через PCG

  • леса, кусты, траву;
  • камни, скалы, объекты окружения;
  • дороги, тропы, распределение материалов;
  • динамическую растительность;
  • контент на основе RVT, высоты, наклона, масок.

PCG — это мощный инструмент для больших миров и open-world проектов.

World Partition

World Partition автоматически делит мир на ячейки и подгружает их по мере необходимости. Это позволяет создавать огромные карты без ручного управления уровнями.

Преимущества:

  • автоматическая подгрузка;
  • поддержка стриминга;
  • идеально для open-world;
  • работает с PCG, Water, Landmass.

Советы по работе с продвинутыми инструментами

  • Используйте Landmass для дорог и рек вместо старых Splines.
  • Применяйте RVT для смешивания материалов и травы.
  • Используйте PCG для больших лесов и экосистем.
  • Для огромных миров — VHM + World Partition.
  • Nanite Landscape подходит для скал и каньонов.

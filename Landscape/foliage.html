Foliage – растительность

В настройках - editor preferences поставить галочку в Procedual Foliage

Можно размещать растительность в большом количестве на поверхности ландшафта, травы, деревья и тд.

Вкладка foliage – переносим туда объекты с content browser. с помощью кисти ЛКМ растравляем в случайном порядке, можно отключить объект который не хотим расставлять через галочку.

Так же указывается количество ассетов в группе, это получается как один объект, не нагружает систему, не как расставлять по одному мешу тысячи раз.

Если расставлять объекты по отдельности, потребуется много ресурсов, количество запросов на отрисовку будет равна количеству объектов, а в foliage всё как один объект.

Большое количество инстанцев, то есть собраны в один объект, при отрисовки каждого элемента нет необходимости вызвать каждый элемент отдельно.

Так бы было миллион запросов, а так достаточно одного запроса, по одному адресу – увеличивает производительность, большое количество объектов не нагружает систему – повторяемые копии объекта называются инстансы, отлично подходят для создания растительности в большом объеме.

Инструменты растительность

Paint – размещение или удаление растительности

Brush size – размер кисти

Paint density – плотность объектов

Erase density – плотность стирания

Single instance mode – размещать по одной копии объекта

Place in current level – размещать на данном уровне если их несколько

Filters – где можно и нельзя размещать растительность

Reapply – применить настройки к инстансу, у каждого инстанса есть свои настройки, например расстояние друг от друга.

Выделяем нужный инстанс и меняем нужные настройки radius – радиус разбора.

Так же можно полностью поменять mesh объекта и абсолютно все инстансы изменятся.

Select – выбрать конкретный объект из всего инстанса и перемещать его, разворачивать, масштабировать и удалять

Select all – выбрать все объекты

Select invalid – выбрать неправильно размещённые объекты, например выше уровня или ниже поверхности.

Lasso select – выделить объекты в конкретной области, и манипулировать с выделенными объектами.

Flll – заливка полностью всего пространства, но весь рельеф нельзя заполнить.

Например, создать большую площадь и заполнить его инстансами. Вся площадь двигается вместе с инстансами, так же с удалением площади удалится и инстанс.

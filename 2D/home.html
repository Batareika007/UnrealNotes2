новая тема

создать игры в 2D

для 2D нужны ассеты, это будут обычные пиксельные картинки, есть два вида
  1. индивидуальные картинки, например если есть 5 анимаций движений будет 5 отдельных картинок
  2. sprite sheet - одна большая картинка, в которй по порядку нарисованны все анимации

создаем в проекты набор ассетов, по папкам распределяем где какие, и загружаем все картинки которые у нас есть для проекта, все они будут как текстуры. Если надо меняем название, например T_PlayerHealthBar

создать спрайты из текстур

для 2Д игр, не испульзутю текстры на прямую, а используют спрайт, для этого нужен плагин: Paper2D - должен быть по умолчанию, дополнительный плагин который нужен PaperZD

каждой текстуре нужно поменять значение на 2D Pixels тогда картинка будет выглядить намного лучше - зайти в каждую текстуру

но есть способ проще, правой кнопкой на тектстуру и выбрать - Apply Paper2D Texture Settings - это автоматически выставит настройки которые нам нужны

выставить на сцену

чтоб выставлять на уровень, нужно превратить текстуру в спрайты

индивидуальная картинка:

пкм на текстуру - sprite action - create sprite

теперь этот объект можно просто выставлять на сцену как актор

sprite sheet:

нужно разделить весь комплект на отдельные спрайты

выбираем текстуру и так же

пкм на текстуру - sprite action - extract sprite

движок автоматом разделит объекты, зависит от оригинала, иногда это нужно сделать вручную тогда в настрока меняем на grid и выставялем высоту и ширину каждого объекат

сделать анимации

для анимирования справйтов используется flipbook, выделяем нужные спрайты, ПКМ - Create flipbook

изменить размер

если нужно поменять размер, используем значение Pixels per unit

можно как каждому отдельному спрайту, или во всём проекте назначить всем спрайтам подрять, чем меньше значение, тем больше элемент

Создать персонажа

для 2д есть специальный класс для создания персонажа

лучше всего увеличить персонажа чтоб он поместился в капсулу, меняем Pixels per unit, но тогда это значение нужно менять у всех ассетов в проекте

камера

в 2д нам не нужен свет, облака, туман, отмосфера и тд. можно сразу во всем проекте убрать автоэкспозицию и дать значение на 10 но зависит от проекта, или же эти значение написать дать камере игрока

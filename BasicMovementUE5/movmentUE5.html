Далее создаём IMP - input mapping context для нашего персонажа, можно также указать его имя

И там прописываем для каждого созданного инпута как им управлять, для прыжка и управлением камерой, можно оставить простые настройки А вод для движения немного сложнее 

    Для движение вперед выбираем swizzle input axis values, и в Order - YXZ 
    Для движение налево - Negate и оставляем все галочки
    Для движения назад - выбираем и swizzle и negate вместе и все галочки на месте
    А вот для движение направо можно ничего не указывать, он автоматом подхавит управление, то есть не важно какую клавишу мы укажим, он пойдем направа

Далее создаем персонажа если у нас его ещё нет, blueprint class - character

Задаем mesh может быть всё что угодно, хоть квадрат. Создаем SpringArm - что-то типа селфи палки, пока она выделена - добавляем камеру и она автоматом прикрепится .

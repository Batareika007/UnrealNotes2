Плагины, которые я использую

Actor Palette

Плагин позволяет загружать дополнительный уровень и переносить объекты из него прямо на сцену. Это значительно удобнее, чем искать варианты в Content Browser. После установки в меню появляется отдельная панель, где можно выбрать нужный уровень.

Actor Locker

Позволяет «заблокировать» актор на сцене, чтобы его нельзя было случайно выбрать или переместить. Очень полезно при работе со сложными сценами.

Water plugin - experimental but can test it

Экспериментальный плагин для создания океанов, рек и озёр. Можно использовать для тестов, но в продакшене требует осторожности.

Niagara

Основная система визуальных эффектов в Unreal Engine. Используется для частиц, огня, дыма, магии, взрывов и любых VFX.

NiagaraFluids

Дополнение к Niagara, позволяющее создавать симуляции жидкостей, газов и volumetric‑эффектов.

Modeling - Modeling Tools Editor Mode

Набор инструментов для редактирования геометрии прямо в Unreal Engine. Обычно включён по умолчанию. Позволяет быстро править модели без внешних 3D‑редакторов.

Lyra Animations

Набор анимационных инструментов и систем из проекта Lyra. Используется для продвинутой анимации персонажей.

Необходимые модули:

  • Animation Locomotion Library
  • Animation Warping
  • Orientation Warping

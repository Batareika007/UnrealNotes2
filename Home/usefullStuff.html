Разные полезности в анрил

  1. Приватность - не отправлять личную информацию.
    Privacy → usage data: ip, isp (провайдер), континент, страна, город, аккаунт ID, название проекта
  2. Color piker - при выборе цвета можно сохранять пресеты и делать наборы
  3. Highlight objects under mouse cursor - подсвечивать объекты в viewport при наведении мыши.
  4. Убрать Developer folder из билда:
    Project Settings → Packaging → Directories to never cook → добавить Developers.
  5. Reference viewer - нажать на любой объект и узнать с чем он связан
  6. Заменить повторяющиеся ассеты:
    выбрать ассеты → Asset Actions → Replace References → выбрать, какой оставить → Consolidate Assets.
    unreal автоматически подставит нужный reference для всех объектов
    Инструмент заменяет всё: материалы, меши, ссылки
  7. Global Asset Picker (Ctrl + P) — глобальный поиск ассетов, можно фильтровать, не зависим от папок
  8. Property matrix - редактировать за один раз несколько объектов
  9. Выбрать все одинаковые акторы на сцене — Shift + E.
    Middle Mouse — переместить точку (pivot).
  10. Merge Actors — выбрать акторы → меню Actor → Merge.
    Можно сделать копию, создать новый объект на уровне или инстанс.
  11. Game Mode — нода Restart Player at Player Start возвращает игрока на стартовую позицию.
  12. Target Point — удобный объект для указания позиции: Look At Target, Spawn SFX и т.д.
  13. Play in Game: Shift + F1 — показать курсор.
    во время симуляции К сохранить изменения обхектов
  14. Platform → Cooking on the Fly — используется при разработке и тестировании. В экспорте галочку нужно убрать.
  15. Редактировать уровни из другого уровня:
    Window → Levels → Add Existing Level → редактируем его без открытия.
  16. Обновить ноды в акторе — Refresh All Nodes
    Если есть проблемы в логике, часто помогает.
  17. Макрос — то же, что Collapse Nodes, но глобальный.
    Blueprint Macro Library → Actor — доступен в любых акторах
    Но иногда проще просто сделать collaps
  18. Functions — выполняются моментально, без задержки.
  19. Unreal Diff Tool — сравнить два объекта, увидеть различия, если они похожи.
  20. Widget — с помощью Ctrl можно двигать pivot (одуванчик).
  21. physics - обновляется каждый фрейм, если скорость предмета больше скорости между фреймами, то мрдет провалится
    так же физика просчитывает где объект должен находится в следующем фрейме, и если есть что-то что ему мешает
  22. initial Life Span - сколько живет актор, в панели Details можно назначить через какой время актор уничтожится, 0 - будет жить всегда.

Разные полезности в анрил

  1. Приватность - не отправлять личную информацию Privacy – usage data: ip, isp (провайдер), континент, страна, город, аккаунт ID, название проекта
  2. Color piker - при выборе цвета, можно сохранить и делать наборы
  3. Highlight objects under mouse cursor - в viewport подсвечивать объекты на которые наведена мыш
  4. Убрать Developer folder из билда – project settings – packaging – directories to never cook... developers
  5. Reference viewer - нажать на любой объект и узнать с чем он связан
  6. Заменить повторяющиеся ассеты - выбираем одинаковые ассеты - asset action – replace references - выбираем какой оставить - consp;odate assets – unreal автоматически подставит нужный reference для всех объектов Инструмент заменяет абсолютно всё, может поменять материал так и меш
  7. Global asset picker , ctrl + p -- глобальный поиск ассетов, можно фильтровать, не зависим от папок
  8. Property matrix - редактировать за один раз несколько объектов
  9. Выбрать все одинаковые акторы на сцене - shifr + E Middle mouse – move point
  10. Merge actors - выбрать акторы, верху в меню actor – merge, можно сделать копию, создать новый на уровне, сделать инстанс
  11. Game mode - нода Restart Player at Player Start - вернуть игрока на стартовую позиуию
  12. Target point - объект удобен для разных указаний в какой части что-то сделать - look at target, spawn sfx и тд
  13. Play in game “shift + F1” - show cursour/ во время симуляции “К” сохранить изменения обхектов
  14. Platform – cooking on the fly - во время разработки и тестирования, в экспорте убрать галочку
  15. Редактировать уровни из другого уровня, без необходимости его открывать: windows – levels –add exsisting level и его редактируем
  16. Обновить ноды в акторе - refresh all nodes - если есть проблемы в логике, решает их
  17. Макрос - тоже самое как colaps node. Создать независимвй Blueprint – blueprint macro library – actor - вызывается в любих акторах. Но иногда проще просто сделать collaps
  18. Functions - функция которая происходит моментально, без времени
  19. Unreal Diff tool - сравнить два объекта на различия, увидеть изменения если они одинаоквые
  20. Widget - с помощью ctrl двигаем одуванчик

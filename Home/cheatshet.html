Шпаргалка

Каждый юнит = 1 см

Три стрелки на объекте называются gizmo.

Центр стрелок на объекте — pivot point.

Meshes:

  • Static Mesh — статичный меш, двигается как одна целая модель.
  • Skeletal Mesh — меш с подвижными частями.

Skybox — огромная полая сфера.

Временно переместить pivot point:
на одну из стрелок alt + СКМ
или нажать на сам pivot point + СКМ

End — опустить выделенный объект на поверхность (если нет пересечений с другими коллизиями).

Surface Snapping — размещать объекты по поверхности с учётом угла.

Surface offset – расстояние между объектами, будет прилегать на расстояние которое указали а не впритык

Blueprint — Flow Control - базовые управляющие ноды

F11 – Immersive Mode (во весь экран)

Content Browser – удобно фильтровать ассеты

Выделить объект и подлететь к нему — клавиша F

Зажать Alt при выделенном объекте — вращение камеры вокруг него.

ПКМ – Browse to Asset или Ctrl + B – найти объект в Content Browser.

LOD — Level of Detail

Система снижает детализацию объекта при удалении от камеры. В настройках объекта можно задать, насколько уменьшается количество полигонов.

Создание референса на игрока

Создать камеру для быстрого перемещения: Ctrl + 0–9.

Выбрать сохранённую камеру: 0–9.

Пилотировать свет: ПКМ по источнику света → Pilot “... Light” или Ctrl + Shift + P.

Настройка Input

Чтобы инпут работал во время паузы — включить галочку “Execute When Paused”.

Level

A collection of objects and their properties that define an area of gameplay.

Volumes

Heightmap

Sculpt

World Settings

Можно настроить Kill Z — уничтожает всё, что падает ниже указанной высоты.

По дефолту можно настроить прямо в настройках мира

Чтобы отключить — убрать галочку “Enable World Bounds Checks”.

Physics

Angular Damping — сопротивление вращению.

Delta Seconds

get world delta seconds промежутоу времени между фреймами.
чтоб объекты не зависили от фреймов, а двигались независимо, всегда умнодаем результат на delta seconds
это дает нам одинковое движение не зависимо от кадров
меньше FPS - больше delta seconds и наоборот
тем самым компенсирует и добавляет скорость или уменьшает

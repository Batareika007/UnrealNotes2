Это очень базовые команды для Git. Раздел не обновляется и служит только как архивная заметка.

GIT

GIT — используется для подключения проекта к GitLab.

Source Control → Connect to Source Control

Provider → Git

В GitLab создаём простой репозиторий, копируем URL‑ссылку и вставляем её в Unreal Engine в поле URL. После подключения автоматически создаётся первый коммит.

Initialize project with Git — инициализирует проект под Git, ждём завершения.

Accept Settings — применяем настройки.

Indicator Source Control — зелёный значок означает, что проект подключён к системе контроля версий.

После каждого изменения в Content Browser появляется красная иконка — это значит, что объект изменён и готов к коммиту.

Чтобы сохранить изменения: Source Control → Submit to Source Control → пишем комментарий к коммиту.

После коммита можно отправить изменения на сервер командой Push.

В gitlab создаем простой репозиторий, копируем URL-сылку и вставляем в unreal engine в URL -- это сразу создает первый комит

Initialize project with Git – инициализирует с гит и ждем результата

Accept settings – применяем все свойства

Tener source control – зеленый значит подключен к source control

После каждого изменения, будет появляется красная галочка на объектах в content browser – значит что зафиксированы изменения.

выбираем source control – submit to source control и пишем комит – что именно изменилось

теперь можем запушить изменения на сервер с помощью команды push

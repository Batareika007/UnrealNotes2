Настройка проекта (Project Settings)

В разделе Project Settings находятся все основные параметры проекта: описание, карты, режимы игры, настройки движка, физики, рендера, платформ и упаковки. Ниже приведены ключевые категории и их назначение.

Description

Информация о проекте: название, версия, компания, автор, номер билда. Пример версии: 0.0.1a.

Gameplay Tags

Система тегов, используемая для логики игры: состояния персонажа, типы оружия, взаимодействия, эффекты. Позволяет создавать гибкие и масштабируемые механики.

Maps & Modes

Настройка базового игрового режима (GameMode) и стартовых карт:

  • Editor Startup Map — карта, открывающаяся в редакторе.
  • Game Default Map — карта, запускаемая в игре.
  • Default GameMode — основной игровой режим.

Packaging

Настройки упаковки проекта: путь сборки, формат файлов, использование Pak-файлов, исключение контента редактора. Здесь же включается Full Rebuild и выбираются карты для Cooking.

Supported Platforms

Выбор платформ, которые проект должен поддерживать. Отключение лишних платформ ускоряет сборку и уменьшает размер проекта.

Target Hardware

Настройки под целевое оборудование: мобильные устройства, ПК, консоли. Можно снизить параметры до минимально допустимых для экономии ресурсов.

Engine

Общие настройки движка: рендеринг, физика, навигация, звук, оптимизация, память.

AI System

Настройки искусственного интеллекта: поведение, perception, навигация, оптимизация AI.

Collision

Настройка каналов коллизии: создание новых каналов, изменение поведения (Block/Overlap/Ignore). Используется для взаимодействия объектов, оружия, трассировки лучей.

Cooker

Определяет, как проект будет запекаться (Cooking): режимы, оптимизация, исключение контента, работа с картами.

Crowd Manager

Настройки поведения толпы (AI Crowd): избегание столкновений, плотность, скорость реакции.

General Settings

Общие параметры проекта: шрифты, настройки материалов, поведение незапечённых теней, UI-параметры.

Hierarchical LOD (HLOD)

Система оптимизации больших уровней. Позволяет автоматически объединять объекты в группы и создавать LOD-модели для дальнего расстояния.

Navigation Mesh

Настройки навигационной сетки (NavMesh): размер ячеек, точность построения, высота/ширина проходов. Чем меньше размер ячейки — тем точнее навигация, но выше нагрузка на память и время генерации.

Physics

Настройки физики: гравитация, поведение объектов, скорость снарядов, взаимодействие тел. Можно включить или отключить Chaos Physics.

Rendering

Настройки рендера: HDR, Lumen, Nanite, сглаживание, тени, LOD-оптимизация, качество материалов. Важно для производительности и визуального качества.

Streaming

Управление стримингом ассетов: подгрузка текстур, уровней, моделей. Влияет на производительность и время загрузки.

Platforms

Отдельные настройки под каждую платформу: Windows, Android, iOS, Linux, консоли. Здесь можно отключить ненужные платформы.

Rendering → Lens Flares

Настройки бликов и эффектов камеры.

  • Bloom — эффект яркого свечения. Отключение делает свет более мягким.
  • Auto Exposure — автоматическая экспозиция (адаптация глаза). Для VR рекомендуется отключать.

Перемещение камеры

ЛКМ + мышь
Поворот камеры влево/вправо и движение вперёд/назад по осям X–Y.

ПКМ + мышь
Свободный обзор вокруг точки — вращение камеры без перемещения.

ЛКМ + ПКМ + мышь или колёсико мыши
Перемещение камеры вправо/влево/вверх/вниз (оси X–Z).

Мышь + WASD
Перемещение как в игре: вперёд/назад/влево/вправо.

Мышь + QE
Подъём и опускание камеры (ось Z).

Мышь + ZC
Временный зум — приближение и отдаление, пока удерживаются клавиши.

Работа с объектами

F
Выбрать объект и переместить камеру к нему (Focus on Selected).

Alt + ЛКМ
Орбита вокруг выбранного объекта.

Alt + V
Привязать pivot к ближайшему вертексу объекта.

Ctrl + B
Открыть в Content Browser папку, где находится выбранный ассет.

Ctrl + E
Открыть Blueprint или редактор ассета выбранного объекта.

Работа с нодами (Blueprint / Material Editor)

Q — выровнять ноды по горизонтальной линии.

Shift + S — выровнять по нижнему краю.

Shift + W — выровнять по верхнему краю.

Shift + D — выровнять по правому краю.

Ctrl + A — выровнять по левому краю.

Alt + ЛКМ
Удалить соединение между нодами.

Ctrl + Drag
Перетянуть и подключить ноду к другой.

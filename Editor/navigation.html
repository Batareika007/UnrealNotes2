ЛКМ + мышь

Поворачивать право лево, движение вперед назад

Движение право лево верх низ

Оси X Z

ПКМ + мышь

Крутится на одной точки везде

Мышь + WSAD

Движение как в игре, вперед назад право лево

Мышь + QE

Движение вверх вниз

Мышь + ZC

Зум ближе дальше, временный пока держим клавиши

F

Выбрать предмет и направится к нему при нажатии

Alt + мышь

Крутится вокруг объекта

ALT + V

Прикрепить pivot к одной из точек вертексов объекта

Ctrl + B - открыть в браузере место выделенного объекта

Ctrl + E - открыть blueprint

Редактировать выравнивать ноды

Q - выровнять по линии

Shift + S - выравнивать по нижней части

Shift + w - выравнивать по верхней части

Shift + d - выравнивать по правому краю

Ctrl + a - выравнивать по левому краю

Alt + leftMause - отключить

Ctrl + drag - перетянуть и подключить

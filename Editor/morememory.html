Увеличение памяти для движка (Texture Pool)

Unreal Engine использует выделенный объём видеопамяти для текстур — Texture Pool. Если в проекте много высококачественных материалов, движок может начать снижать качество текстур, чтобы уложиться в лимит. В этом случае можно увеличить размер пула.

Проверка текущего размера

Введите команду в консоль:

r.TexturePoolSize

Команда покажет, сколько мегабайт выделено под текстуры в данный момент.

Установка постоянного значения

Чтобы задать фиксированный размер пула, отредактируйте файл DefaultEngine.ini:

Unreal Projects\ProjectName\Config\DefaultEngine.ini

Добавьте строки:

[/Script/Engine.RendererSettings] r.Streaming.PoolSize=2000

Значение 2000 означает 2000 МБ (2 ГБ). Выбирайте размер в зависимости от видеокарты — слишком большое значение может вызвать проблемы на слабых системах.

Download steam SDK

Documentation to steam https://dev.epicgames.com/documentation/en-us/unreal-engine/online-subsystem-steam-interface-in-unreal-engine?application_version=5.1

Download sdk from steam and past it in unreal folder – the Unreal Enging version of the game And change the version to the version of SDK from steam, for now it’s 1.62 Open file Steamworks.build.cs and change the version to 1.62 or hire

не обновлять версию SDK, иначе проект выдают ошибку при упаковке, используй только встроенную версию

Обновить файл DefaultEngine.ini в проекте

открыть defailtEngine.ini и добавить строки, данные есть в официальной документации эпиков

[/Script/Engine.GameEngine] +NetDriverDefinitions=(DefName="GameNetDriver", DriverClassName="OnlineSubsystemSteam.SteamNetDriver", DriverClassNameFallback="OnlineSubsystemUtils.IpNetDriver") [OnlineSubsystem] DefaultPlatformService=Steam [OnlineSubsystemSteam] bEnabled=true SteamDevAppId=480 [/Script/OnlineSubsystemSteam.SteamNetDriver] NetConnectionClassName="OnlineSubsystemSteam.SteamNetConnection"

Проверка на работоспособность

Приложение Steam должно быть включено

Если всё сделано правильно запускаем проект в отдельном окне stand alone game нажимаем shift + tab, окошко Steam полжно показаться на экране

Добавить ачивки в стим

на старницы стима добавляем ачивки для нашей игры и только после этого редактируем defailtEngine.ini

[OnlineSubsystemSteam] bEnabled=true

SteamDevAppId=480 – убрать 480, написать твой номер сборки! или использовать 480 для

Achievement_0_Id= название твоей ачивки на странице steam

Steam SDK

Unreal Engine уже содержит встроенную версию Steam SDK. Не обновляйте SDK вручную — это вызывает ошибки при упаковке проекта.

Документация Epic Games: Online Subsystem Steam

Download sdk from steam and past it in unreal folder – the Unreal Enging version of the game And change the version to the version of SDK from steam, for now it’s 1.62 Open file Steamworks.build.cs and change the version to 1.62 or hire

Hе обновлять версию SDK! Используйте только встроенный Steam SDK, который идёт вместе с версией Unreal Engine.

Настройка DefaultEngine.ini

Открыть файл DefaultEngine.ini в проекте и добавьте строки:

[/Script/Engine.GameEngine] +NetDriverDefinitions=(DefName="GameNetDriver", DriverClassName="OnlineSubsystemSteam.SteamNetDriver", DriverClassNameFallback="OnlineSubsystemUtils.IpNetDriver") [OnlineSubsystem] DefaultPlatformService=Steam [OnlineSubsystemSteam] bEnabled=true SteamDevAppId=480 [/Script/OnlineSubsystemSteam.SteamNetDriver] NetConnectionClassName="OnlineSubsystemSteam.SteamNetConnection"

Проверка на работоспособность

Steam-клиент должен быть запущен

Запустите проект в режиме Standalone Game. Нажмите Shift + Tab. Если всё настроено правильно — появится оверлей Steam.

Добавление достижений (Achievements)

Сначала создайте достижения на странице вашей игры в Steamworks. После этого добавьте их в DefaultEngine.ini.

[OnlineSubsystemSteam] bEnabled=true

SteamDevAppId=480 — замените 480 на ID игры. 480 можно использовать только для тестирования.

Achievement_0_Id= идентификатор достижения из Steamworks (название ачивки)

После этого можно добавлять логику в Blueprints или C++.

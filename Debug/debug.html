Базовый набор команд и инструментов

Debug — инструменты отладки

Раздел Debug включает команды консоли и инструменты редактора, которые помогают анализировать производительность, искать проблемы, проверять нагрузку на систему и оптимизировать проект. Этот раздел будет расширяться по мере добавления новых тем.

Основные команды консоли (Cmd)

Команды для отображения статистики производительности:

  • stat fps — показывает количество кадров в секунду.
  • stat unit — отображает время работы Game, Draw, GPU, Frame.
  • stat game — показывает нагрузку игровых систем.

Ограничение максимального FPS

Команда для установки максимального количества кадров:

t.maxfps 30

Полезно для тестирования производительности, мобильных устройств или экономии ресурсов.

Delta Time

Delta Time — это время между кадрами. Чем стабильнее Delta Time, тем плавнее работает игра.

Увеличение видеопамяти для стриминга текстур

Команда для изменения размера пула текстур:

r.Streaming.PoolSize 3000

Значение указывается в мегабайтах. Например, 3000 = 3 GB видеопамяти под стриминг текстур.

Настройка через конфигурационные файлы

Можно задать PoolSize через конфиг:

Engine → Config → Windows → WindowsEngine.ini

В этом файле можно включить параметры для увеличенного использования RAM и VRAM.

Анализ ассетов: вес, зависимости, оптимизация

Unreal Engine предоставляет инструменты для анализа размера ассетов и их связей.

Reference Viewer

Показывает, на какие ассеты ссылается объект, и какие ассеты ссылаются на него.

  • ПКМ по объекту → Reference Viewer

Size Map

Отображает, сколько места занимает ассет на диске и в памяти.

  • ПКМ по объекту → Size Map

Это помогает находить тяжёлые ассеты, оптимизировать текстуры, модели и материалы.

Debug sphere

draw debug sphere - рисует сферу для дебага (если и другие фигуры)

  • center - где в мире будет нарисована сфера
  • draw debug arrow - нарисовать стреллу
  • draw debug cordinate system

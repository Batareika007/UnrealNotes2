Data Layers

Data layers работает только в с World Partioin - он включен по умолчинию в больших мирах Open World

для простой карты basic или Empty Level нужно конвертировать этот уровень в world partition:

Tools - world partition - convert level

выбираем уровень который нужно сконвертировать и ждём.

у нас будут два уровня, первый оригинальный а второй с world partiton (в название уровня будет добавлен _WP)

создать Data Layer

ПКМ в разделе world - Data layer. и даешь ему название

windows - World partition - Data Layers Outliner - здесь будет список всех Data Layers который мы хотим добавить на уровень, и туда добавляем наш созданый Data Layer, или же можно прям там его создать

выбираем какие акторы будут прям в нем, можешь добавлять и удалиять любые акторы из Data layer

Blueprint

вызываем ноду Get Data Layer Manager - из неё вытягиваем Set Data Layer Runtime State - указываем сылку на Data Layer и его положение, если он не загружен, загружин или активен.

очень базовые настройки

Create First-Person Character

В Unreal Engine каждый персонаж управляется своим Player Controller. Контроллер получает ввод (клавиатура, мышь) и передаёт команды персонажу.

Action Mapping — действия по нажатию кнопки (Jump, Fire, Interact).
Axis Mapping — оси для плавного ввода (мышь, движение вперёд/назад).

1. Создаём персонажа и контроллер

Создаём два Blueprint:

  • BP_Character — сам персонаж;
  • BP_PlayerController — контроллер, который будет им управлять.

Затем в настройках карты указываем, какой персонаж и контроллер используются по умолчанию.

2. Настройка камеры

Камера должна находиться на уровне глаз персонажа. Если капсула имеет высоту 88 (половина), то полный рост — 176 см. Камеру поднимаем примерно на уровень глаз.

Для вида от первого лица обычно используют угол обзора около 60°.

3. Управление камерой мышью

В Input выбираем события Mouse X и Mouse Y. Это не переменные — это Axis Mapping.

Добавляем поворот:

  • Yaw — поворот влево/вправо (всего персонажа);
  • Pitch — взгляд вверх/вниз (только камера).

4. Важный момент: Pitch должен вращать только камеру

Если оставить настройки по умолчанию, капсула персонажа будет наклоняться вверх/вниз — это неправильно для FPS. Поэтому Pitch должен быть отключён у капсулы и включён только у камеры.

В компоненте камеры включаем галочку Use Pawn Control Rotation.

Теперь:

  • Yaw вращает весь персонаж;
  • Pitch вращает только камеру.

Это правильная настройка для First-Person персонажа.

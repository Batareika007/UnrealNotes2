Enumerators (Enum)

Enum — это список заранее известных вариантов, из которых переменная может принимать только одно значение. Это удобно, когда нужно описать состояние, режим, тип или категорию объекта: например, состояние персонажа, дни недели, тип оружия, пункты меню.

Enum заменяет множество отдельных переменных и делает логику чище и безопаснее.

1. Создание Enum

В Content Browser создаём Blueprint Enumeration.

Внутри добавляем список всех возможных вариантов.

2. Использование Enum в Blueprint

Создаём переменную и выбираем тип — наш Enum. Теперь переменная может принимать только одно значение из списка.

3. Передача Enum через Events

Enum можно передавать как параметр в Custom Event. Это удобно для переключения состояний или режимов.

4. Сравнение Enum

Enum можно сравнивать между собой. Например, проверить: персонаж стоит, бежит или прыгает.

5. Switch on Enum

Самый удобный способ работать с Enum — нода Switch on Enum. Она автоматически создаёт выходы для всех вариантов Enum.

Для каждого варианта можно задать свою логику.

Events Dispatcher

создает список слушателей, в момент когда что-то происходит, каждый слушатель в списке будет уведомлен и запустит совй евент

то есть один объект уведомляет другие когда что-то происходит

Call - запускает

вызывает события диспечера (актор который кричит другим)

bind - подключить слушателя

актор который должен слушать подписывается на диспечера

unbind - отключить одного

удаляет подписку - актор который слушал, больше не слушает

unbind all - отключить всех

актор который кричит, прекращает кричать, очищает список слушатетель

event - создать событие

актор который слушает, вызывает события

assign - подключить и создать

быстрое полключение и вызов (bind + event) в акторе который слушает

Widget внутри

после создания виджета, user interface - widget blueprint

нужно указать главнй компонент в котором будут все другие

можно на весь экран поставить картинку и только она там и будет

Overlay - достаточная ограниченная, но простая и меньше жрёт ресурсов

Canvas Panel - универсальная панель, можно свободно выставлять в любом месте компоненты, очень тяжелый, обычно создается одни раз на всё игру

вместо canvas можно использовать к примеру size box

в виджете нажимаем на любой добавленный компонент правой кнопкой, и можно оборачивать в новый из списка

Важно

не писать слишком много логики в виджете, всё что не связано с визуальной частью виджета лучше переносит в другие объекты,то есть не давать виджету слишком много контроля над игрой

редактор снаряда

projectile снаряд

для снаряда есть специальный компонент движения - Projectile Movment

управление снарядом

одно из главный значение - скорость

  • Initila Speed - начальная скорость, с которой объект стартует, применяется один раз при старте
  • Max Speed - максимальная скорость снаряда, 0 - безлемита, даже если начальная скорость будет выше максимальной, скорость всё равно обрежется до максимальной
  • Projectule Gravity Scale - как будет работат гравитация, добавить или вообще убрать для лайзера например
  • initial Life Span - в панели detile всего актора, отлично для снарядов чтоб сами уничтожались и не захламляли память

если мы хотим через логику задавать разную скорость для снарядов, от Initila Speed = 0 и отключаем Initial Velocity in Local Space

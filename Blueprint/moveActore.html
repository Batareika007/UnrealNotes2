Actor Movement — перемещение акторов

В Unreal Engine есть несколько способов перемещать акторы. Каждый из них работает по‑разному и используется в разных ситуациях: телепортация, плавное движение, движение по локальным координатам, перемещение с учётом коллизий и т.д.

1. Set Actor Location

Set Actor Location — перемещает актор в указанную мировую позицию (World Space).

Это прямое указание: «поставь объект в точку X/Y/Z».

Используется для:

  • телепортации персонажа;
  • перемещения объектов по сцене;
  • спавна и позиционирования;
  • быстрого перемещения без анимации.

Sweep

Опция Sweep включает проверку коллизий по пути к новой точке.

  • Включено — объект «скользит» по пути и остановится, если встретит препятствие.
  • Выключено — объект переместится напрямую, игнорируя всё.

Sweep полезен, если нужно:

  • двигать объект с учётом препятствий;
  • избежать прохождения сквозь стены;
  • делать корректное перемещение без телепортации.

Teleport

Опция Teleport определяет, как физика реагирует на перемещение.

  • Включено — объект телепортируется, физика НЕ применяется.
  • Выключено (по умолчанию) — физика учитывается, объект может столкнуться, отскочить и т.д.

Пример: Если телепортировать Skeletal Mesh с включённой физикой — он может «взорваться» или разлететься. Teleport = true предотвращает это.

2. Add Actor World Offset

Add Actor World Offset — двигает актор на указанное расстояние в мировых координатах.

Это не установка позиции, а смещение:

  • вектор (100, 0, 0) — сдвинет объект на 100 единиц вперёд по миру;
  • вектор (0, 0, 50) — поднимет объект на 50 единиц вверх.

Используется для:

  • плавного движения;
  • анимации объектов;
  • движения платформ;
  • скриптового перемещения.

Важно: направление движения НЕ зависит от того, куда смотрит объект.

3. Add Actor Local Offset

Add Actor Local Offset — двигает актор по его собственным локальным осям.

То есть:

  • вперёд — это направление, куда смотрит объект;
  • вправо — его локальная ось Y;
  • вверх — его локальная ось Z.

Пример:

  • Если персонаж повернулся — движение вперёд всегда будет в сторону взгляда.
  • Если дверь повернута под углом — локальное движение будет следовать её ориентации.

Используется для:

  • движения персонажа;
  • анимации объектов, которые должны двигаться «в своём направлении»;
  • выдвижных ящиков, дверей, лифтов;
  • любых объектов, где важно направление.

4. Когда что использовать?

Нода Когда использовать
Set Actor Location Телепортация, мгновенное перемещение, позиционирование.
Sweep Перемещение с учётом препятствий.
Teleport Перемещение объектов с физикой без «взрывов».
Add Actor World Offset Плавное движение по миру, анимации, платформы.
Add Actor Local Offset Движение в направлении объекта (вперёд/назад/вверх).

5. Важные нюансы

  • Если объект не двигается — проверь коллизии (может блокироваться).
  • Для персонажей лучше использовать Add Movement Input, а не Set Location.
  • Для физики лучше использовать Add Force / Add Impulse.
  • Для анимаций дверей/ящиков — Local Offset + Timeline.

Эти ноды — основа любого движения в Unreal Engine. Понимание разницы между ними делает работу с акторными трансформациями намного проще.

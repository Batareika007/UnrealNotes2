Bluprint Nodes

Get world location - даёт данное положение игрока на сцене. По трем осям x y z

Relative location - даёт расстояние от родителя, если в самом блюпринте отодвинуть от начала

Get world rotation - данные наклонения обекта на сцене по трем осям

Get relative rotation - данные наклона от сцены

Get world scale - размер scale mesha

Get world Transform - показывает все параметры - location, rotation, scale

Spawn object - открываем blueprint всего уровня

Spawn actor from class

Spawn actro from class - можно добавить любому блюпринту и выбрать где будет спавнится новый обект

Destroy actor - удалить актор с игры, тем самым очистить память , указывам какой актор удалить

Is valid ? - проверяеть валидность ноды, если существует проблема, программа не будет продолжаться чтоб не совершать будущих проблем

Get all actors of Class - получать доступ к акоторм на сцене, возвращает нам массив, далее этот массив можно превратить в переменную и ей задать различные манипуляции

Tags - каждому объекту можно добавить теги, тем самым дополнять функционал

Пример применения

Set timer by event - будет запускать евент с интервалом, так же можно задать ему что повторялся бесконечно... кстати вместо event tick Надо будет создавать кастомный эвент

Clear timer - останавливает интервал таймер, надо только указать имея функции которую надо остановить

Forward Vector - показывает в какую сторону персонажа направляется то есть куда лицом по осям, xyz от -1.00 до 1.00

Lerp - имеет два входных данных, этом может быть любые данные, номер, вектор, поворот и тд. Разницу между ними это альфа - номер от 0-1 в процентах, то есть и этим легче управлять удалённо, например с глазиком Альфа 0 - 0% // альфа 0.5 - 50%

Event Hit

My Comp - PrimitiveComponent  - с каким из компонентом в акторе было совершено столкновении, например если внутри актора несколько компонентов
Other - какой актор столкнулся
Other Comp - компонент другого актора, который столкнулся
Self Moved - булеан, кто был причиной удара, true если актора двигался сам, false если в актор попали
Hit Location - место, где было столкновение
Hit Normal - направление поверхности 
Normal Impulse - сила удара / импульса
Hit - вся информация о столкновении в виде структуры, можно вытащить “break” там будет подробная информация о столкновении

Event end play - почему актор перестал существовать на сцене

Event Receive Draw HUD

Работает только в HUD - нарисует маленьки квадра на который можно кликнуть

Switch Nodes - рабоатет как if else получает входные данные и выводит результат

Do N - действие может выполниться ограниченное повторы, после чего перестает срабатывать, только если будет сделан рестарт. Он не выполняется подряд, а как только я хочу выполнить действие

DoOnce - делает что-то только один раз, можно начать сразу с запрета, на действие и перезапустить чтоб снова сработал

Flip Flop - каждый раз вызывает то одно ветвь то вторую, можно отслеживать какая ветвь была вызвана через булеан

For loop - сделает повторные действия столько раз сколько указано, одновремено, при больших числах может быть очень требователен

ForLoopWithBreak - тоже самое, только есть возможность остановить последовательность, например при достижение индекса

Gate - проверка открытия ворот, по умолчанию открыты, в Enter заходит логика, если ворота открыты она проходит дальше в exit если закрыты, то в exit не проходит, и есть возможность открыть и закрыть воврота, так же начать с того чтоб ворота были закрыты, и менять статус ворот

MultiGate - Это переключатель между несколькими выходами, который по очереди или случайно активирует один из них при каждом вызове.

Sequence - запустить несколько веток по порядку, однако скорей всего это будет выполнено практически одновременно -- это визуальный и логический помощник структурирования, чтоб было проще искать и поинмать логику

WhileLoop - повторять бесконечные действия пока True, и это может привести к зависанию игры. WhileLoop — это инструмент для мгновенных вычислений, не для визуальных или пошаговых действий. Он нужен, когда ты хочешь быстро пройти по логике, но тебе не важна задержка или визуальный эффект.

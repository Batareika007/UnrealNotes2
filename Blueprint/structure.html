Structure

Очень полезная штука для сохранения любого количества данный.

Структура - группа с разными типами переменных, для определенной цели, как пример может быть данные персонажа, имя, уровень, здоровье и тд.

Добавляем разные переменные

Так же можем назначить значения по умолчанию

Получать и изменять данные в event graph

Добавляем структурру как переменную, выставляем в граф get и пкм - break structure

В breake мы получаем данные, а чтоб изменять данные нужно выставить переменную как set и далее сделать make structure, или раскрыть весь список set – split

Чтоб изменять конкретные свойстава а не все подряд, используем set members

Базовые классы

Классы

В Unreal Engine каждый объект в игре основан на одном из базовых классов. Понимание этих классов помогает правильно строить логику, управлять персонажами и организовывать проект.

Actor

Базовый объект, который можно разместить на уровне или заспаунить во время игры. Это самый часто используемый класс.

  • Используется для создания механик и объектов: лампочки, двери, фонарики, триггеры.
  • Имеет Location, Rotation и Scale.
  • Содержит Tick Event — вызывается каждый кадр.
  • Акторы можно спавнить и уничтожать во время игры.

Pawn

Объект, которым можно управлять — вручную или через ИИ. Это «пешка», в которую игрок или AI может «вселяться».

  • Имеет Location, Rotation, Scale.
  • Не получает управление автоматически — это нужно настроить заранее.
  • Получает команды от контроллера (Player Controller или AI Controller).
  • Игрок может вселиться в Pawn (например, машина, самолёт, робот).

Character

Расширенный Pawn с готовой логикой передвижения и скелетной анимацией.

  • Умеет ходить, прыгать, приседать, падать.
  • Имеет встроенную капсулу коллизии.
  • Содержит Skeletal Mesh и систему движения Character Movement.

Player Controller

Класс, который принимает ввод игрока (клавиатура, мышь, геймпад, тач) и передаёт команды Pawn или Character.

  • Контролирует только одну пешку одновременно.
  • Не имеет физического тела в игре — это чистая логика.

GameMode Base

Определяет правила игры: когда начинается матч, когда заканчивается, условия победы, подсчёт очков и т.д.

  • У каждого уровня может быть свой GameMode.
  • Выбирается в World Settings → GameMode Override.
  • Создаётся как обычный Blueprint Class.

Actor Component

Логический компонент, который можно использовать в разных блюпринтах. Помогает не дублировать код.

  • Подходит для механик урона, движения, взаимодействия.
  • Можно добавлять в разные акторы.
  • Экономит время и делает проект чище.

Scene Component

Компонент, который имеет позицию, поворот и масштаб. Используется как «точка» или «узел» для других компонентов.

  • Все объекты на сцене основаны на Scene Component.
  • Примеры: лестница, балка, пустой объект‑маркер.

Создание Blueprint Class

В Content Browser → ПКМ → Blueprint Class — выбираем один из базовых классов (Actor, Pawn, Character, GameMode и т.д.).

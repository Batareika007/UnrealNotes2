переменные / Data types

Variables – переменные – сведенья о blueprint – создаем новую переменную «+»

Variable types:

Boolean

true / false

Byte (8 bits)

числено от 0 - 255 , занимает мало памяти, только положительные, нет отрицательных чисел

Integer (32 bits)

целое число без остатка, положительно и отрицательное от -2 до +2 билиона -2.147.483.648 до 2.147.483.647, использует много памяти

Integer64 (64bits)

очень большое число с 16 нулями, очень много памяти -9,233,372,036,854,775,808 до 9,233,372,036,854,775,807

Float (32 bits)

дробное значение, занимает больше памяти чем integer

Name

текст обозначающее имя чего либо, по размеры самый легкий, мало памяти, не может быть изменен,не переведен, нельзя манипулировать, не зависит от регистра (заглавные и строчные буквы), не использует пробелы

String

текст который может изменять размер, шрифт и тд. Занимает больше памяти, стандарт для использования

Text

текст который может быть переведен на любой язык - локализация, занимает очень много памяти, используется в виджетах

Vector, Rotator, Transform

структура для геометрии

Vector - X,Y,Z

Rotator - Pitch, Yaw, Roll

Transform - Location, Rotation, Scale

Structure - содержит много разныз переменных в одном месте, как к пример hit result

Interface - имеет свой функционал который мы можем написать

Object Types - Object and class References, and soft References

Enum - списос простых данных, которым мы можем пользоваться

Array

массив в блюпринтах очень полезен, при создании ноды, пишем Array и он покажет нам все возможности что можно сделать с массивом, вот некоторые из них:

создаём массив и мы можем получи из него данные, найти данный, узнать длину массива, найти последний элемент, проверить если массив содержит данные, добавить в массив, удалить из массива по индексу или содержанию, очистить весь массив, соединить два массива.

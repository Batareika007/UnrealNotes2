переменные

Variables – переменные – сведенья о blueprint – создаем новую переменную «+»

Variable types:

Boolean

true / false

Byte

числено от 0 - 255 , занимает мало памяти

Integer

целое число без остатка, положительно и отрицательное от -2билиона до +2 билиона, использует много памяти

Integer64

очень большое число с 16 нулями, очень много памяти

Float

дробное значение, занимает больше памяти чем integer

Name

кусок текста обозначающее имя чего либо, по размеры самый легкий, мало памяти, не может быть изменен, переведен, нельзя манипулировать

String

кусок текста который можно изменять размер, шрифт и тд. Занимает больше памяти

Text

кусок текста который может быть переведен на любой язык, локализация, занимает очень много памяти, если тебе нужна локализация то ты пользуешь текст.

Vector

3Д структура которая хранит x, y, z типа float

Array

массив в блюпринтах очень полезен, при создании ноды, пишем Array и он покажет нам все возможности что можно сделать с массивом, вот некоторые из них:

создаём массив и мы можем получи из него данные, найти данный, узнать длину массива, найти последний элемент, проверить если массив содержит данные, добавить в массив, удалить из массива по индексу или содержанию, очистить весь массив, соединить два массива.

В designer устанавливаем канвас, место интерфейса, можно выбрать под любой размер монитора или мобильного телефона.

В сам канвас добавляем кнопки и текск и тд

Anchor points - якорь, где будет располагаться объект ,растягиваться ,перемещаться в зависимости от разрешения экрана

Vertical / Horizontal BOX - контейнеры в которых элементы автоматически встают один за другим горизонтально или вертикально

чтоб сортировать вертикально и горизонтально можно использовать Grid Panel

Show / Hide UI to Viewport - Показать в игре

На BP уровня, создаем ноду и указываем виджет будет отображаться

Далее надо создать переменную

Назавем переменную например UI_text или Меню - смотря что мы создаем, и дальше можно манипулировать, напрмер при нажатии кнопки высвечивать виджеи и убирать

так же можно напряму сразу выставлять

Input Modes - выставлять на экран разные кнопки и взаимодействовать с ними, есть три разные позиции,

Set Input Mode Game only - инпут и продолжать управлением персонажа но без мышки

Set Input Mode Game And UI - инпут и управлвать персонажем и мышкой -- управление мышью будет рабоать только для UI, управление персонажем будет не возможно

Set Input Mode UI Only - инпут, не управлять персонажем а только мышкой Обязательно надо подключить player controller,

Чтоб была видна мыш, подключаем ноду show mouse cursor - к ней подключаем player controller

И в блюпринте добавлять ноду именно к этой кнопке Set keyboard Focus

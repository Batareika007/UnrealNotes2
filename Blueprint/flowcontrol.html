Flow Control

Branch

условие которые выводит в true или false

Быстрое создание, держим ‘B’ и нажимаем ПКМ

Do Once

сделает только один раз за всё время запуска кода Start closed - не будет сразу запущенно если это позволяет

DoOnce - делает что-то только один раз, можно начать сразу с запрета, на действие и перезапустить чтоб снова сработал

Do N

ограничить выполнения действий N раз, то есть нажать на кнопку 3 раза и после этого действие не будет срабатывать. В примере, выполнится три раза каждый тик, и больше не будет выполнять, при нажатие на F снова выполнится три раза

Do N - действие может выполниться ограниченное повторы, после чего перестает срабатывать, только если будет сделан рестарт. Он не выполняется подряд, а как только я хочу выполнить действие

Do Once Multi

сделать один раз из списка, есть несколько дверерй но открыть можно только одну

Delay

запустит действие спустя назначеное время

Retriggerable Delay

запустить действие спустя назначенное время, но если запустить действие опять, то время обнулится и начнется с начала

Flip flop

при каждом вызове будет менять из одного параметра на другой Is A - будет вызывать значение булион если это вариант A-true или false

Flip Flop - каждый раз вызывает то одно ветвь то вторую, можно отслеживать какая ветвь была вызвана через булеан

Gate

как открытые ворота. Мы можем выбрать если они изначально закрыты или открыты.
Enter - запускает ноду, если ворота открыты то пойдет дальше, если закрыты, то дальше не пойдет
Open - открывает ворота, теперь enter - будет работать
Close - закрывает ворота - Enter не работает
Toggle - закрывает и открывает ворота

Gate - проверка открытия ворот, по умолчанию открыты, в Enter заходит логика, если ворота открыты она проходит дальше в exit если закрыты, то в exit не проходит, и есть возможность открыть и закрыть воврота, так же начать с того чтоб ворота были закрыты, и менять статус ворот

Multi Gate

похожи на гейт только больше параметров При вызове этой ноды, каждый раз будет производится действие по порядку которые мы назначим
Значени loop и random - будет заново повторять все действие и рандомально запускать, но рандомально запускать каждый раз всю последовательность и только потом начинать заново.
Мы не сможем рандомально вызвать первый и опять первый, а только когда он пройдет все данные, тогда рандомально начнёт заново.
Reset - обнуляет gate и начинает с нуля.

MultiGate - Это переключатель между несколькими выходами, который по очереди или случайно активирует один из них при каждом вызове.

Sequence

будет проигрывать один за другим всё что мы подключим, всё производится мгновенно без задержек, но по очереди

Sequence - запустить несколько веток по порядку, однако скорей всего это будет выполнено практически одновременно -- это визуальный и логический помощник структурирования, чтоб было проще искать и поинмать логику

Switch

проверить входные данные и вызвать поток по результату

For Loop

производит постоянные действия несколько раз, зависит от индекса который мы задаём. Loop body - то что будет отражаться, Completed - то что произойдёт в конце нашего лупа.

For loop - сделает повторные действия столько раз сколько указано, одновремено, при больших числах может быть очень требователен

ForLoopWithBreak

тоже самое, только есть возможность остановить последовательность, например при достижение индекса

While loop

тоже самое что и for loop, но он происходит до тих пор пока правдивое состояние Так же есть что будет происходить каждую итерацию и что будет по окончанию. Нельзя ставить состояние которое будет крутиться бесконечно, получим ошибку бесконечного лупа

WhileLoop - повторять бесконечные действия пока True, и это может привести к зависанию игры. WhileLoop — это инструмент для мгновенных вычислений, не для визуальных или пошаговых действий. Он нужен, когда ты хочешь быстро пройти по логике, но тебе не важна задержка или визуальный эффект.

Switch Nodes - рабоатет как if else получает входные данные и выводит результат

Vectors — работа с векторами в Unreal Engine

Вектор — это набор из трёх значений (X, Y, Z), который описывает:

  • позицию в мире;
  • направление;
  • скорость;
  • расстояние;
  • движение;
  • повороты и ориентацию объектов.

Вектора — основа любой 3D‑игры. Понимание их работы позволяет управлять движением, физикой, камерой, ударами, AI и многим другим.

1. Расстояние между двумя векторами

Если вычесть один вектор из другого, мы получим разницу по осям X, Y, Z.

Это не расстояние, а вектор смещения.

2. Vector Length — длина вектора

Чтобы получить реальное расстояние, применяется теорема Пифагора. В UE для этого есть нода Vector Length.

Она возвращает длину вектора — то есть прямое расстояние между точками.

Есть также Vector Length Squared — расстояние в квадрате.

Это быстрее и полезно для оптимизации, когда не нужно точное расстояние.

3. Get Distance To — расстояние между объектами

Самый простой способ узнать расстояние между двумя актёрами.

Возвращает расстояние в сантиметрах.

4. Горизонтальное и вертикальное расстояние

Чтобы узнать расстояние только по одной оси:

  • по Z — вертикальное расстояние;
  • по X/Y — горизонтальное расстояние.

5. Vector Normalize — нормализация

Нормализованный вектор — это вектор длиной 1 (единичный вектор).

Он показывает только направление, без расстояния.

Используется для:

  • направления движения;
  • направления взгляда;
  • расчёта силы удара;
  • пушек, трассеров, лучей;
  • AI-навигации;
  • физики.

Чтобы получить движение, нормализованный вектор умножают на скорость.

6. Forward / Right / Up vectors

Каждый объект в UE имеет три основных направления:

Forward Vector

Показывает, куда объект смотрит (ось X).

Right Vector

Показывает правую сторону объекта (ось Y).

Up Vector

Показывает вверх (ось Z).

Все эти вектора всегда нормализованы (длина = 1).

Их можно получить также через Get Rotation → Get Forward/Right/Up Vector.

7. Rotation From X Vector

Позволяет повернуть объект так, чтобы его ось X указывала в нужное направление.

Используется для:

  • поворота пушек;
  • ориентации эффектов;
  • вращения камер;
  • направления AI.

8. Find Look At Rotation

Поворачивает объект так, чтобы он смотрел на другой объект.

Нужно указать:

  • кто смотрит;
  • на кого смотрит;
  • какой объект получает этот поворот.

10. Итог

  • Вектора описывают позицию, направление и движение.
  • Vector Length — расстояние.
  • Normalize — направление.
  • Forward/Right/Up — ориентация объекта.
  • Look At Rotation — повернуть объект к цели.
  • Velocity — скорость персонажа.

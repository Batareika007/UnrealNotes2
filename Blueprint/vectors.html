Узнать расстояние между двумя векторами, просто путём вычитанем один у другого, но это дает нам растояние в xyz

Vector length

Чтоб получить прямое растояние до обекта, надо применить теорему пифагору, для этого есть функция Vector Length

Так же можно получить результат в квадрате, для использования теоремы

Get Distance To

Узнать расстояние между двумя объектами - Get Distance To

Узнать только вертикально или горизонтально растояние до обекта

Vector Normalize - найти значение Normalize это может быть от 0-1 - то есть мы создаем новый вектор значения которого по каждой оси будет максимум 1

Нужен для расчета направления, куда смотерть, куда идти и тд..

Так же нужен для правильного рачета удара, силы, движения и к результат умнажаем или прибавляем для получение нужной цели.

Forward vector - всегда указывает куда смотри персонаж и у этого вектора всегда значение будет normalize то есть 0 - 1 по каждой оси xyz, forward vector показывает направление по оси X

Right vector - указывает правую сторону персонажа, так же всегда normalize 0-1 по оси Y

Up vector - указывает направление вверх так же от 0-1 по оси Z

Все эти векторые так же можно получить из ноды Get Rotation – там в результате будет все три параметра и можно вызвать каждый из них.

С помощью данных вектора можно крутить другие обекты назначая им те же кординаты, это происходи с помощью Rotration from X vector

Создать объект чтоб наблюдал за персонажем

Find Look at Rotation - будет вертеть объект и постоянно указывать на положение другого объекта

  • ** выбираем тот кто будет следить
  • ** выбираем за кем будет следить
  • И в конце назначаем какой объект будет двигаться за следящим

Velocity - скорость персонажа

Будет указывать скорость персонажа по всем осям, то есть скорость по движению по xyz

Скорость всегда сантиметр в секунду

Так же скорость может быть отрицательно, если движение назад

Результат всегда будет в трех параметрах xyz

Может превратить в один с помощью vector length

Но! К обычной скорости, будет прибавляться скорость в прыжке и показатель будет увеличен Тогда можно подключить только x y

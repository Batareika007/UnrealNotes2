действие событий происходит в случаях:

  • Triggered - нажатие и держание
  • Started - нажал на кнопку
  • Ongoing - нажать и держать определенное время (в игре когда нужно держать кнопку нажатой )
  • Canceled - не держал достаочно времени нажатую кнопку
  • Completed - отпустить кнопку
  • Action Value - зависит от вида кнопки, есди это bool или axis
  • Epapsed Seconds - сколько времени держишь кнопку нажатой (идет отчет)
  • Triggered Seconds - время между срабатываниями триггеров, например если есть триггре pulse
  • Input Action - ссылка на себя

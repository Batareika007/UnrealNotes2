Input Action

Input - Input action

создать базовое указание ввода, называем AI_действие

Value Type :

  • Digital (bool) - одноразовое нажатие кнопки, отправдяем информацию если кнопака нажата или нет
  • Axis 1D (float)
  • Axis 2D (Vector 2D) - для движение мыши имеет два вектора X Y
  • Axis 3D (vector)

Triggers , Modifiers

добавляение здесь триггеров будет влиять на все кнопки которые мы добавим в InputMappingContext

  • Triggers - управление повидеинем, лугика активации, какие будут действии, нажать, удерживать, как часто выполнять действи при держании и тд.

    • так же при выборе будет открыты разные настройки поведения каждого триггера

    например если выбрать pulse то можно задать интервал срабатывания, держим кнопку, срабатывает Triggered и каждый промежуток времени в секундах которые мы задали, будет происходит эвент

  • Modifiers - изменять входные данные, как изменится ввод при нажатии

    • увеличить или уменьшить чувствительность мыши, повернуть или инвертировать ось, и тд.

    Negate - перевернуть значение, альтернатива в блюпринте умнажать на -1 но лучше сделать в контроллере

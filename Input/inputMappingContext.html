Input Mapping Context

input - Input Mapping Context

собираем все вводные кнопки значение которые собрали и назначаем каждому кнопку называем imc_название управление то есть чем и как хотим управлять

в разделе Mapping добавлять вводные данные которые мы создали

к каждому значению можно бодавить значения, здесь мы может редактировть индивидуально каждую клавишу, если не задано общее значение в input Action

  • Triggers - управление повидеинем, какие будут особености при действии

    • так же при выборе будет открыты разные настройки поведения каждого триггера

    например если выбрать pulse то можно задать интервал срабатывания, держим кнопку, срабатывает Triggered и каждый промежуток времени в секундах которые мы задали, будет происходит эвент

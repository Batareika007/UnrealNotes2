Свет и туман

Освещение и туман — ключевые элементы атмосферы сцены. Unreal Engine предоставляет несколько типов источников света и гибкие настройки тумана, позволяющие создавать реалистичную или стилизованную визуальную среду.

Типы источников света

  • Directional Light — основное «солнце» сцены. Светит параллельными лучами, подходит для дня/ночи.
  • Point Light — точечный источник света, светит во все стороны. Аналог лампочки.
  • Spot Light — направленный световой конус. Используется для фонарей, прожекторов.
  • Rect Light — прямоугольный источник света. Даёт мягкое, ровное освещение, подходит для окон, экранов, неона.

Отключение Lumen

Если нужно отключить глобальное освещение Lumen:

r.DynamicGlobalIlluminationMethod 0

Включить обратно: r.DynamicGlobalIlluminationMethod 1

Туман (Fog)

Основные параметры Exponential Height Fog:

  • Fog Density — плотность тумана. Чем выше значение, тем гуще туман.
  • Fog Height Falloff — скорость уменьшения плотности по высоте. Чем выше значение, тем быстрее туман «рассеивается» вверх.
  • Start Distance — расстояние от камеры, на котором туман начинает проявляться.

Создание туманного света (Volumetric Fog Light)

Чтобы получить красивый «кинематографичный» свет в тумане (лучи света, объёмный свет), выполните следующие шаги:

1. Post Process Volume

  • Infinite Extent (Unbound) — применяет эффект ко всей сцене.
  • Temperature — настройка цветовой температуры (по желанию).
  • Exposure — при необходимости установить Manual для стабильной яркости.

2. Exponential Height Fog

  • Enable Volumetric Fog — включить объёмный туман.

3. Любой источник света

  • Volumetric Scattering Intensity — увеличить значение (обычно 5–20), чтобы световые лучи были заметны в тумане.

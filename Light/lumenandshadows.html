Красивый свет с Люмен

в редакторе включает следующие параметры

выбрать directX для люмена

platform - windows

Target RHI - DirectX12

DirectX 11 & 12 (SM6) - check

Rendering

lumen

Global Illumination - Lumen -- очень требовательно и дорого для игр

Reflection - Lumen -- очень требовательно и дорого для игр

Use Hardware Ray Tracing when available - check (будет только работаеть если видео карта поддерживает RTX) но если есть проблемы в разработке лучше отключить

Ray Lighing Mode - Surface Cache

Software Ray Tracing mode - Detail Tracing - можно поставить на Global Tracing тогда будет более щедящий режим для производительнсоти, но менее эфективный визуально

Shadow Map Method - Virtual Shadow Maps - более детальные тени

Ray Traced Shadows - работет только вместе с включеным Hardware Ray Tracing, акуратные тени но очень требовательны к железу

support Hardware Ray Tracing - это главное значение если, без него вообще не будет работать Ray Tracing если выключить

Allow Static Lighing - обязательно включить если мы хотим запекать свет
